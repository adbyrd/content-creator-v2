Below is the complete memberProfile.js file for your Member Profile page. It combines built‑in member data with custom fields from the FullData collection, and enforces paid‑member access via a backend pricing plans check.


Important Setup Notes
UI Element IDs
Make sure your page contains elements with exactly these IDs:

#profilePhoto (image)

#coverPhoto (image)

#firstName (text)

#lastName (text)

#nickname (text)

#company (text)

#websiteUrl (link or text – if link, the code sets both text and link)

#companyDescription (text or rich text)

(Optional) #errorMessage (text) – add this if you want to display errors on the page.
