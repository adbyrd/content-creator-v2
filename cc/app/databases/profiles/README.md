# Profiles

Collection Name: “CC Profiles” (id: ccProfiles)

Member ID (id: memberId): The email used to register the account
Company Name (id: companyName): The name of the company.
Company Email (id: companyEmail): The company email address
Description (id: description): A description of the company
URL (id: url): The URL of the company website
Category (id: category): The business category the company classifies as.
Sub Category (id: subCategory): The sub business category the company classifies as.
Customer Type (id: customerType): The customer type the company serves (B2C, B2B, etc).
Zip Code (id: zipCode): The zip code of the company headquarters.
Company Logo (id: companyLogo): A url to the company logo
